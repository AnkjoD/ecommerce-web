# 🛠️ Homura Shop - Admin Dashboard

This is the administrative control panel for Homura Shop. Built with Vite, React, and Shadcn UI, it provides a high-performance interface for managing products, orders, users, and shop settings.

## 🚀 Key Features
- **Product Management**: Create, update, and manage product variants and categories.
- **Order Tracking**: Monitor real-time sales and order statuses.
- **User Management**: Oversee customer accounts and permissions.
- **Analytics**: Visualized shop performance (Revenue, Orders, Best sellers).
- **Security**: Role-based access control with JWT authentication.

## 🛠️ Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Components**: Shadcn UI + Radix UI
- **State Management**: TanStack Query (React Query)
- **Forms**: React Hook Form + Zod validation

## 🏃 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Setup Environment
Ensure you have the `.env` file configured.

### 3. Start Development Server
```bash
npm run dev
```

## 🏗️ Folder Structure
- `src/components`: Reusable UI components (Shadcn).
- `src/hooks`: Custom React hooks for data fetching.
- `src/pages`: Main dashboard views and layouts.
- `src/apis`: Axios-based API client for the backend.
